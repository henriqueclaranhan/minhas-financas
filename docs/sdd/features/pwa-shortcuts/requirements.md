# PWA Shortcuts Requirements

## Goal

Provide direct access to the application's primary destinations from the installed PWA icon context menu.

## Acceptance Criteria

- The manifest exposes shortcuts for Home, Transactions, Planning, and Invoices.
- Every shortcut opens the canonical application route within standalone mode.
- Shortcut metadata uses concise Portuguese labels consistent with the default manifest language.
- Every shortcut uses a dedicated PNG representation of the same Lucide icon shown for that destination in the main navigation.
- Existing PWA installation, icons, and offline behavior remain unchanged.
