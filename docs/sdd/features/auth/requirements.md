# Requirements: Auth

## Feature Overview
Handle user authentication allowing users to securely log in or register.

## User Stories
- As a User, I want to create an account using my email and password so I can access my finances.
- As a User, I want to log in using my Google account for convenience.

## Acceptance Criteria
- [x] Users can register with email and password.
- [x] Users can log in with email and password.
- [x] Users can sign in with Google.
- [x] Validation errors (e.g., weak password, wrong email) are displayed clearly.\n
