# Requirements: Dashboard

## Feature Overview
A main dashboard providing a snapshot of the user's financial status.

## User Stories
- As a User, I want to see my total income, expenses, and current balance so I know my financial standing at a glance.
- As a User, I want to see a chart of expenses by category to understand where my money is going.

## Acceptance Criteria
- [x] Dashboard displays total balance, income, and expenses cards.
- [x] Displays a chart grouping expenses by category.
- [x] Displays each category icon in the expense chart legend and tooltip.
- [x] Shows a list of recent transactions.\n
- [x] The monthly expense card opens the current-month expense breakdown.
- [x] Every dashboard summary card is navigable and uses the same top-right chevron without supplementary action copy.
