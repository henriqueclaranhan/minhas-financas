# Requirements: Internationalization (i18n)

## Feature Overview
Implement a global internationalization sweep across the entire application to support multiple languages seamlessly.

## User Stories
- As a User, I want to use the application in my native language so that I can better understand my financial data.
- As a User, I want to switch languages in the settings and have the UI update immediately.

## Acceptance Criteria
- [x] All hardcoded UI strings are replaced with translation keys.
- [x] Application provides Portuguese (pt-BR) and English (en) dictionaries at minimum.
- [x] Language selection is persisted globally and updates the UI instantly.
