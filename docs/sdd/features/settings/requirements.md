# Requirements: Settings

## Feature Overview
Main settings hub to configure application preferences.

## User Stories
- As a User, I want to change the app language.
- As a User, I want to toggle between light and dark themes.
- As a User, I want data management actions on a dedicated screen so that destructive and backup operations are easy to identify.

## Acceptance Criteria
- [x] Language selector (PT, EN, ES, etc).
- [x] Theme toggle switch.
- [x] Preferences are persisted.
- [x] Settings exposes a translated Data destination alongside Profile and Privacy Policy.
- [x] Export, import, progress, and clear-data actions live on the nested `/settings/data` screen.
