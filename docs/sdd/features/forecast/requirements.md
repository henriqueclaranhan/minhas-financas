# Requirements: Forecast

## Feature Overview
Predict future financial balance based on planned and recurring expenses.

## User Stories
- As a User, I want to see how my balance will look in the coming months based on my fixed expenses.

## Acceptance Criteria
- [x] Chart displaying projected balance over time.
- [x] Configurable forecast parameters (e.g., how many months ahead).
- [x] Projection dates support Month, Year, and custom Period while planned-value toggles remain next to the chart.
- [x] Large projected balances use localized compact currency labels on the vertical axis without clipping or consuming the chart area.
- [x] Tooltips and data tables retain exact, non-compacted currency values.
